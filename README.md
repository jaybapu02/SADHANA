# Sadhana

**Effort + commitment = Daily progress**

## 📌 About the Project

Sadhana is a Django-based study tracking and productivity application designed to help students stay consistent and focused. It enables users to track study sessions, monitor performance, and build disciplined habits over time.

## 🚀 Features

* 📊 Study session tracking
* 👨‍👩‍👧 Parent-child connection system
* 📈 Dashboard with analytics
* 🔔 Notifications & reminders
* 🔐 User authentication system
* ⏱️ Time tracking for productivity

## 🛠️ Tech Stack

* Backend: Django (Python)
* Frontend: HTML, CSS, JavaScript
* Database: SQLite

## ⚙️ Installation & Setup

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/Sadhana-2.0.git
   cd Sadhana-2.0
   ```

2. Create virtual environment

   ```bash
   python -m venv myenv
   source myenv/bin/activate  # Windows: myenv\Scripts\activate
   ```

3. Install dependencies

   ```bash
   pip install -r requirements.txt
   ```

4. Run migrations

   ```bash
   python manage.py migrate
   ```

5. Start server

   ```bash
   python manage.py runserver
   ```
